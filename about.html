About

Birdnet analyzer is een tensorflow model getraind om vogelzang te herkennen. Normaal is deze tool te gebruiken via de command line. Deze website is gemaakt als gebruiksvriendelijker alternatief voor de command line tool. Met de website kan een geluidsopname geanalyseerd worden en word een overzicht gegeven. Voor gebruik van de website zie de use pagina op de website.
